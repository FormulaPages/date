# date

![CircleCI](https://circleci.com/gh/FormulaPages/date.svg?style=shield&circle-token=:circle-token)

Calculate serial number with year, month and day.

## Syntax

```js
DATE(year, month, day)
```

## Install

```sh
npm install --save formula-date
```

## Usage

```
var date = require('formula-date');
date( 2015, 10, 6 )
```
